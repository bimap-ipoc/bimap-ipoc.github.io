安裝步驟

安裝步驟

下載安裝包

解壓安裝包並執行安裝腳本

#!/bin/bash

USERNAME=bimap
PASSWORD=1qaz2wsx
IP=$(hostname -I  | tr -d '[:blank:]')

echo 'start => 安裝相依套件 (要等一下)'
sudo apt-get update
sudo apt install mysql-server nginx openjdk-17-jdk zip ssh sysstat nmon -y &> /dev/null

sudo mysql < db_init.sql

echo 'start => 安裝 keycloak 模組包'
echo 'keycloak 預設帳號：bimap 預設密碼：1qazxcv'
sudo dpkg -i ./keycloak.deb
echo 'start => 更新 keycloak ip'
sudo sed -i "s/change_me/$IP/" /usr/bin/keycloak/conf/keycloak.conf
sudo systemctl restart keycloak


echo 'start => 安裝 bimap 模組包'
sudo dpkg -i ./ipoc-backend/bimap-ipoc-backend-v1.4.1.deb
sudo dpkg -i ./ipoc-frontend/bimap-ipoc-frontend-v1.3.9.deb


echo 'start => 更新 ipoc yaml 的 host ip'
sudo sed -i "s/{{change_to_host_ip}}/$IP/" /etc/bimap-ipoc/config.yml


echo 'start => 安裝 influxdb'
wget https://dl.influxdata.com/influxdb/releases/influxdb2-2.7.1-amd64.deb
sudo dpkg -i influxdb2-2.7.1-amd64.deb
sudo systemctl restart influxdb

echo 'start => 安裝 influxdb cli'
wget https://dl.influxdata.com/influxdb/releases/influxdb2-client-2.7.3-linux-amd64.tar.gz
tar xvzf ./influxdb2-client-2.7.3-linux-amd64.tar.gz
sudo cp ./influx /usr/local/bin/

echo 'start => 初始化 influxdb'
influx setup --org master --bucket oss_metrics --username $USERNAME --password $PASSWORD --force

echo 'start => 列出 influxdb token'
INFLUX_TOKEN=$(influx auth ls | awk '/bimap/{print $4}')
sudo sed -i "s/{{change_influxdb_token}}/$INFLUX_TOKEN/" /etc/bimap-ipoc/config.yml

echo 'start => 啟動 ipoc 前端'
sudo systemctl restart nginx

echo 'sudo systemctl restart bimap-ipoc'

echo '確認 bimap-ipoc 服務啟動正常 ，就用 curl 初始化 keycloak'
echo 'curl -X GET http://localhost:8700/api/v1/set-sso-by-ipoc'

配置系統所需的參數和環境變數

驗證配置

  1. 確認設定：確保所有基本和進階設定已正確配置並保存。
  2. 測試連接：測試伺服器和數據源的連接，確保所有數據能夠正常上傳和解析。
  3. 檢查儀表板：進入儀表板頁面，確認數據顯示正確，所有指標和圖表都能夠正常工作。

配置 Grafana (如有需要)

如果需要使用 Grafana 來進行數據可視化，請進行以下配置：

  1. 取 Grafana API Key
curl -X POST -H "Content-Type: application/json" -d '{"name":"apikey", "role": "Admin"}' http://admin:admin@localhost:3000/api/auth/keys
  1. Grafana DataTable Plugin 設定
grafana-cli plugins install briangann-datatable-panel
systemctl restart grafana-server
cp -rp briangann-datatable-panel /var/lib/grafana/plugins/
chown -R grafana:grafana /var/lib/grafana/plugins/*
  1. 效能數據上傳： 進入「效能數據上傳」選項，配置數據上傳路徑（預設為 /home/sftpuser/master，也可以根據需求修改），確保效能數據能夠正確上傳並解析。
