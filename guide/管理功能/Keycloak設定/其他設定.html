其他設定

Keycloak 設定 - 其他設定

修改自動登出的時限

  1. 點選左側選單的 Realm settings。
  2. 點選中間橫排選單的 Tokens。
  3. 調整 Access Token Lifespan 的設置。
  4. 點擊 Save 保存設定。

修改視窗重新登入時效

  1. 點選左側選單的 Realm settings。
  2. 點選中間橫排選單的 Sessions。
  3. 調整 Login action timeout 的設置。
  4. 點擊 Save 保存設定。

修改預設 admin 帳號

  1. 點選左側選單的 Realm settings。
  2. 點選中間橫排選單的 Users。
  3. 修改 admin 帳號的名稱和密碼。
  4. 點擊 Save 保存設定。
