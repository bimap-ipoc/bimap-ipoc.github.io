Process

系統關聯分析：Process 頁面

頁面介紹

Process 頁面展示系統中各個進程的 CPU 和內存使用情況。用戶可以根據進程名稱和使用率排序，查看資源消耗較大的進程。

操作說明

  1. 進入 iPOC 系統並選擇「系統關聯分析」中的「Process」頁面。
  2. 使用頂部的控制區域選擇不同的進程。
  3. 選擇不同的時間範圍來查看詳細數據。

下拉選項的指標說明

  • CPU 使用率：顯示進程的 CPU 使用情況。
  • 內存使用率：顯示進程的內存使用情況。

圖片說明

圖1：Process 單節點分析頁面，展示選定主機的進程 CPU 和內存使用情況。

  • Group：選擇要分析的群組。
  • Hostname：選擇要分析的主機名稱。
  • Sort：選擇排序方式（如 runtime、cpu_usage、avg）。
  • TOP N：選擇顯示資源使用最高的 N 個進程。
  • 日期選擇器：選擇要分析的日期範圍。
