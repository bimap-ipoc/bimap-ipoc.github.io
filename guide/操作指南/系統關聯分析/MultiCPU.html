MultiCPU

系統關聯分析：MultiCPU 頁面

頁面介紹

MultiCPU 頁面顯示多核 CPU 的使用情況，每個 CPU 核心的使用率分別顯示，並附有歷史數據趨勢圖。這些指標有助於用戶分析和優化多核處理器的性能。

操作說明

  1. 進入 iPOC 系統並選擇「系統關聯分析」中的「MultiCPU」頁面。
  2. 使用頂部的控制區域選擇不同的 CPU 核心。
  3. 選擇不同的時間範圍來查看詳細數據。

下拉選項的指標說明

  • usage：顯示每個 CPU 核心的總使用率。
  • user：顯示用戶模式下的 CPU 使用率。
  • system：顯示系統模式下的 CPU 使用率。
  • steal：顯示被虛擬化環境“偷走”的 CPU 時間。
  • iowait：顯示等待 I/O 操作完成的 CPU 時間。
  • nice：顯示優先級調整後的 CPU 使用率。

圖片說明

圖1：MultiCPU 單節點分析頁面，展示了多主機的 CPU 使用情況和統計數據。

圖2：MultiCPU KPI 分析頁面，顯示各 CPU 核心的不同指標，幫助用戶發現性能瓶頸。

注意事項

  • 確保所選時間段內有足夠的數據，以便進行有效的分析。
  • 根據需要調整時間段和指標，以獲得更多詳細的信息和洞見。
