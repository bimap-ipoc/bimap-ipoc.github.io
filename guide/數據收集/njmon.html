njmon

Njmon 日誌收集說明

1. 安裝 Njmon

下載並安裝 Njmon：

wget http://sourceforge.net/projects/nmon/files/njmon_linux_binaries_v80.zip
unzip njmon_linux_binaries_v80.zip

如果無法解壓，請安裝 zip 命令：

sudo apt install zip

解壓縮後目錄結構如下：

ninstall                      njmon_RHEL8_ppc64le_v80   njmon_SLES15_ppc64le_v80    njmon_Ubuntu21_x86_64_v80
njmon.1.gz                    njmon_RHEL8_x86_64_v80    njmon_SLES15_x86_64_v80     njmon_Ubuntu22_x86_64_v80
njmon_linux_binaries_v80.zip  njmon_SLES12_ppc64le_v80  njmon_Ubuntu20_ppc64le_v80

執行安裝腳本：

chmod +x ninstall
./ninstall

如果沒有完全匹配的系統版本，可以選擇接近的版本，複製到 /usr/local/bin，重新命名為 njmon：

sudo mv ${your_version_filename} /usr/local/bin/njmon
sudo chmod +x /usr/local/bin/njmon

2. 建立 Njmon 日誌存放目錄

mkdir /home/bimap/njmondata

3. 驗證 Njmon 功能

先設定參數進行手動測試：

/usr/local/bin/njmon -M -P -b -t 1 -s 5 -c 2 -f -m /home/bimap/njmondata

檢查生成的檔案：

$ ls /home/bimap/njmondata
hostname_20230327_2200.err
hostname_20230327_2200.influxlp

4. 建立自動化收集腳本

編輯腳本文件：

vim /home/bimap/bin/ipoc_njmon.sh

ipoc_njmon.sh 內容如下：

#!/bin/bash
set -x
export LC_ALL=C

######################################
###             set env            ###
######################################
Host=`hostname`
Seconds=60
Count=60
RetentionDays=30
Yesterday=$(perl -MPOSIX -le 'print strftime "%Y%m%d", localtime(time()-86400)')
ThisHour=$(perl -MPOSIX -le 'print strftime "%Y%m%d_%H%M", localtime(time())')
LogDir="/home/bimap/njmondata"
SentDir=$LogDir/sent
BackupDir=$LogDir/backup
UploadDir="/home/parser/upload/master"

#################################################
########         collect command         ########
#################################################
if [ ! -d $SentDir ]; then mkdir $SentDir; fi
if [ ! -d $BackupDir ]; then mkdir $BackupDir; fi

cd $LogDir
/usr/local/bin/njmon -M -P -b -t 1 -s $Seconds -c $Count -f -m $LogDir > /dev/null 2>&1
sleep 10s
rm -rf *.err
Files=$(ls | grep -v $ThisHour | grep .json)
mv $Files $SentDir

#################################################
########      HourAgoLog to uploadDir    ########
#################################################
scp $SentDir/*.* bimap@10.99.1.55:$UploadDir && mv $SentDir/*.* $BackupDir

#################################################
########          zip and backup         ########
#################################################
cd $BackupDir
if ls *$Yesterday*.json 1> /dev/null 2>&1; then
    tar -czvf `hostname`"-"$Yesterday.tar.gz *$Yesterday*.json && rm -rf *$Yesterday*.json
fi

# 定期將檔案清除
#find $BackupDir/*.tar.gz -maxdepth 1 -mtime +$RetentionDays -type f -delete

exit

授予腳本執行權限：

chmod +x /home/bimap/bin/ipoc_njmon.sh

5. 設定 crontab 每小時執行一次自動化收集腳本

編輯 crontab：

crontab -e

添加以下行：

0 * * * * /home/bimap/bin/ipoc_njmon.sh
